Instructions détaillées

Installation simple en local.

1

Installer Node.js

Installe Node.js LTS sur le PC Windows.

2

Décompresser le ZIP

Place le dossier dans Documents, par exemple G19-Neo-Dashboard.

3

Installer

npm install
4

Lancer

npm start
5

Connecter l’iPhone

Scanne le QR code ou ouvre l’adresse indiquée, puis entre le PIN.

6

Installer sur iPhone

Safari → Partager → Ajouter à l’écran d’accueil.

G19 Neo Dashboard — projet local pour transformer un smartphone en tableau de bord PC.
