Types de configurations

Le logiciel peut être utilisé en mode simple ou avancé.

Configuration minimale

Windows 10/11, Node.js LTS, téléphone sur le même Wi-Fi.

Configuration recommandée

Windows 11, réseau Wi-Fi stable, navigateur Safari/Chrome mobile récent.

Fonctions optionnelles

Météo avec clé OpenWeather. GPU dépendant des compteurs Windows disponibles.

Pages logiciels personnalisées

Une page peut contenir un chemin d’application et 10 raccourcis. Exemple : une page DaVinci personnelle, une page Blender, une page OBS ou une page streaming.
